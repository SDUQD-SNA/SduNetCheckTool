# SduNetCheckTool

为山东大学青岛校区开发的一键式校园网检测工具，欢迎Pull Request && Issue。

## 实现的功能 && TODO

- [x] 网卡检测
- [x] 校园网状态检测
- [x] 系统代理检测
- [x] 网关检测
- [x] 校内网站检测
- [x] 指定IP检测 && 路由跟踪
- [x] 一键修复
- [x] 常用网站检测
